# Sitio Web Académico - Fabián Belmar

Sitio web personal académico creado con HTML/CSS, diseñado para mostrar mi trayectoria investigativa y producción científica.

## 🌐 Vista en Vivo

Visita el sitio: [https://belmarfabian.github.io](https://belmarfabian.github.io)

## 📋 Descripción

Este sitio web académico presenta mi perfil profesional como investigador en ciencias sociales y jurídicas, incluyendo:

- **Publicaciones académicas**: Artículos indexados en Web of Science y Scopus
- **Capítulos de libros**: Contribuciones en obras colectivas
- **Working papers**: Investigaciones en curso
- **Perfil académico**: Enlaces a Google Scholar, ORCID, Scopus
- **CV actualizado**: Curriculum vitae disponible en PDF

## 🎨 Características

- **Diseño minimalista**: Enfoque en contenido con tipografía Georgia
- **Navegación lateral**: Sidebar organizado por secciones
- **Responsive**: Adaptado para múltiples dispositivos
- **Carga rápida**: HTML/CSS puro sin dependencias pesadas

## 🛠️ Tecnologías

- HTML5 (79.9%)
- CSS3 (20.1%)
- R Project (para gestión de contenido académico)

## 📁 Estructura

```
.
├── index.html              # Página principal
├── styles.css              # Estilos del sitio
├── CV FB (Marzo25).pdf     # Curriculum vitae
├── README.md               # Documentación

```

## 🚀 Uso Local

Para visualizar el sitio localmente:

```bash
# Clonar el repositorio
git clone https://github.com/belmarfabian/belmarfabian.github.io.git

# Abrir index.html en tu navegador
cd belmarfabian.github.io
# Doble clic en index.html o usar un servidor local
```
